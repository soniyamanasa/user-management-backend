# User Management Backend

This project is a backend application for user management with authentication and role-based access control.

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Features
- User signup & login
- JWT based authentication
- Role based access (admin/user)
- Protected routes
- Password hashing

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file
4. Add required environment variables
5. Run `node server.js`

## Environment Variables
- PORT
- MONGO_URI
- JWT_SECRET
